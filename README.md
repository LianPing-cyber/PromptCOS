# PROMPTCOS--content-only watermark/verification tool for system prompts!!!

An exciting tool that allows you to watermark your system based on token-level optimization and verify copyright through output content only!


## Quick Start
To get started, install the required dependencies by running:

```bash
pip install -r requirements.txt
```


Check out the example files in the **/prompts** and **/datasets** folders for reference.

Edit the run.sh file to configure the model name, task, and model path. Once configured, you can start the tool.

```bash
bash run.sh
```
